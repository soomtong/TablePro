# Phím tắt

TablePro thiên về bàn phím. Hầu hết thao tác đều có phím tắt, và hầu hết phím tắt menu đều gán lại được.

## Tham Khảo Nhanh

### Phím Tắt Thiết Yếu

| Hành động | Phím tắt |
|--------|----------|
| Thực thi query | `Cmd+Enter` |
| Kết nối mới | `Cmd+N` |
| Mở lịch sử | `Cmd+Y` |
| Cài đặt | `Cmd+,` |
| Quick Switcher | `Cmd+P` |
| Đóng cửa sổ | `Cmd+W` |
| Thoát | `Cmd+Q` |

## Trình Soạn Thảo SQL

### Thực Thi Query

| Hành động | Phím tắt | Mô tả |
|--------|----------|-------------|
| Thực thi query | `Cmd+Enter` | Chạy query tại con trỏ, hoặc chạy tuần tự các câu lệnh đã chọn |
| Giải thích query | `Option+Cmd+E` | Hiển thị kế hoạch thực thi |
| Định dạng SQL | `Option+Cmd+F` | Làm đẹp và chuẩn hóa cú pháp SQL |

<Tip>
`Cmd+Enter` là phím tắt quan trọng nhất. Chạy query tại con trỏ, hoặc chỉ thực thi văn bản đã chọn. Nếu chọn nhiều câu lệnh, chúng chạy tuần tự trong một transaction.
</Tip>

## Phím Tắt Chế Độ Vim

Khi bật chế độ Vim (**Settings** > **Editor** > **Editing** > **Vim mode**), SQL editor dùng phím tắt modal. Badge chỉ báo chế độ hiện trên thanh công cụ.

### Các Chế Độ

| Phím | Hành động |
|------|-----------|
| `Escape` | Quay lại chế độ Normal |
| `i` / `I` | Chèn trước con trỏ / ở đầu dòng |
| `a` / `A` | Chèn sau con trỏ / ở cuối dòng |
| `o` / `O` | Mở dòng mới bên dưới / bên trên |
| `v` | Chế độ Visual (theo ký tự) |
| `V` | Chế độ Visual Line (theo dòng) |
| `:` | Chế độ Command-line |

### Điều Hướng (Chế Độ Normal)

| Phím | Hành động |
|------|-----------|
| `h` / `j` / `k` / `l` | Trái / Xuống / Lên / Phải |
| `w` / `b` / `e` | Từ tiếp / Từ trước / Cuối từ |
| `0` / `$` | Đầu dòng / Cuối dòng |
| `^` / `_` | Ký tự không trống đầu tiên |
| `gg` / `G` | Dòng đầu / Dòng cuối |

### Toán Tử (Chế Độ Normal)

| Phím | Hành động |
|------|-----------|
| `dd` | Xóa dòng |
| `yy` | Yank (sao chép) dòng |
| `cc` | Thay đổi dòng |
| `x` | Xóa ký tự |
| `p` / `P` | Dán sau / trước con trỏ |
| `d` + di chuyển | Xóa với di chuyển |
| `y` + di chuyển | Yank với di chuyển |
| `c` + di chuyển | Thay đổi với di chuyển |

### Chế Độ Visual

| Phím | Hành động |
|------|-----------|
| `d` | Xóa vùng chọn |
| `y` | Yank vùng chọn |
| `c` | Thay đổi vùng chọn |

### Chế Độ Command-line

| Lệnh | Hành động |
|-------|-----------|
| `:w` | Thực thi truy vấn (`Cmd+Enter`) |
| `:q` | Đóng tab |

### Tiền Tố Số Đếm

Thêm số trước phím di chuyển và toán tử: `3j` (xuống 3 dòng), `2dd` (xóa 2 dòng), `5x` (xóa 5 ký tự).

<Note>
Phím tắt Vim chỉ áp dụng trong SQL editor. Không ảnh hưởng data grid hay panel khác. Phím tắt chuẩn như `Cmd+Enter` vẫn hoạt động trong mọi chế độ.
</Note>

## Quick switcher

Quick Switcher (`Cmd+P`) cho phép tìm kiếm và chuyển nhanh đến bất kỳ bảng, view, database, schema, hoặc query gần đây. Hỗ trợ fuzzy matching, gõ `usr` sẽ tìm `users`, `user_settings`, v.v.

| Hành động | Phím tắt |
|--------|----------|
| Mở Quick Switcher | `Cmd+P` |
| Di chuyển kết quả | `Up` / `Down` |
| Mở mục đã chọn | `Return` |
| Đóng | `Escape` |

Kết quả được nhóm theo loại (bảng, view, bảng hệ thống, database, schema, query gần đây) và xếp hạng theo chất lượng khớp khi tìm kiếm.

### Gán lại Phím tắt

1. Mở **Settings** (`Cmd+,`) và chọn tab **Keyboard**
2. Tìm hành động (dùng ô tìm kiếm để lọc)
3. Click vào ô phím tắt
4. Nhấn tổ hợp phím mới
5. Cập nhật ngay trong thanh menu

### Xóa Phím tắt

1. Click vào ô phím tắt
2. Nhấn `Delete` hoặc `Backspace`
3. Phím tắt bị xóa

### Phát hiện Xung đột

Nếu gán phím tắt đã dùng, TablePro hiện hộp thoại xác nhận:

- **Cancel** -- giữ nguyên
- **Reassign** -- chuyển phím tắt sang hành động mới (xóa phím tắt cũ)

<Warning>
Phím tắt hệ thống (`Cmd+Q`, `Cmd+H`, `Cmd+M`, `Cmd+,`) không gán lại được. TablePro sẽ cảnh báo nếu bạn thử.
</Warning>

### Đặt lại về Mặc định

Nhấp **Reset to Defaults** ở cuối cài đặt Keyboard để khôi phục tất cả phím tắt.

### Danh mục Phím tắt

| Danh mục | Ví dụ |
|----------|-------|
| **File** | New Connection, New Query Tab, Close Tab, Save Changes |
| **Edit** | Copy as CSV, Copy as JSON, Export, Import |
| **View** | Toggle Sidebar, Toggle Cell Inspector, Query History |
| **Tabs** | Next Tab, Previous Tab |
| **AI** | Toggle AI Chat, Explain with AI, Optimize with AI |

<Note>
Phím chuyển tab (`Cmd+1` đến `Cmd+9`) theo chuẩn macOS, không tùy chỉnh được.
</Note>

## Mẹo Để Hiệu Quả

### Phím Tắt Dùng Nhiều Nhất

Học 5 phím này trước:

1. `Cmd+Enter` - Thực thi query
2. `Cmd+N` - Kết nối mới
3. `Cmd+Y` - Lịch sử query
4. `Cmd+F` - Tìm
5. `Cmd+C` / `Cmd+V` - Sao chép/Dán

### Mẹo Quy Trình

**Lặp query nhanh**:
1. Viết query trong editor
2. `Cmd+Enter` để chạy
3. Sửa
4. `Cmd+Enter` lại

**Khám phá dữ liệu**:
1. Click bảng trong sidebar
2. Xem dữ liệu trong grid
3. `Cmd+Enter` với query tùy chỉnh
4. `Cmd+Y` tìm query trước đó

**Chỉnh sửa dữ liệu**:
1. Điều hướng bằng phím mũi tên
2. `Enter` để sửa ô
3. Gõ giá trị mới
4. `Tab` chuyển ô tiếp
5. `Cmd+S` lưu tất cả

### Kỹ Thuật Ghi Nhớ

**Nhóm theo ngữ cảnh**:
- `Cmd+` là hành động chính
- Thêm `Shift` cho biến thể (`Cmd+Z` hoàn tác -> `Cmd+Shift+Z` làm lại)
- Thêm `Option` cho phương án thay thế

**Mẫu thường gặp**:
- `Cmd+[chữ cái]` - Hành động thông thường (N=mới, W=đóng, Q=thoát)
- `Cmd+Shift+[chữ cái]` - Biến thể
- Phím mũi tên - Điều hướng
- `Tab` / `Shift+Tab` - Tiếp/Trước

## Khả Năng Truy Cập

### Hỗ Trợ VoiceOver

TablePro hoạt động với macOS VoiceOver:

- Điều hướng bằng `Control+Option+Mũi tên`
- Tương tác bằng `Control+Option+Space`
- Tất cả phần tử UI đều có nhãn

### Điều Hướng Chỉ Bàn Phím

Dùng TablePro hoàn toàn không cần chuột:

1. `Tab` chuyển giữa các phần UI chính
2. Phím mũi tên điều hướng trong phần
3. `Enter` hoặc `Space` kích hoạt control
4. `Escape` đóng dialog/menu

## Khắc Phục Sự Cố

### Phím Tắt Không Hoạt Động

1. **Kiểm tra focus**: Đảm bảo đúng panel có focus
2. **Kiểm tra xung đột**: App khác có thể đang bắt phím tắt
3. **Kiểm tra modifier**: Đảm bảo nhấn đúng phím modifier

### Xung Đột Toàn Hệ Thống

Nếu phím tắt không hoạt động, kiểm tra **System Settings** > **Keyboard** > **Keyboard Shortcuts** tìm xung đột với:

- Mission Control
- Spotlight
- Input Sources
- App Shortcuts
